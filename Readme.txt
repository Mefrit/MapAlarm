

Приложение служит для оповещения пользователя о приближении к некоторой метке на карте.
Приложение позволяет пользователю просматривать карту мира на экране устройства.
Приложение отображает текущее местоположение пользователя на карте.
Приложение позволяет пользователю просматривать, добавлять и удалять собственные метки на карте. Метки имеют описание и приложение позволяет просматривать, добавлять, удалять и редактировать описание меток.
Приложение оповещает пользователя о приближении к установленной им метке на карте стандартным образом, подобно будильнику.
Приложение позволяет настраивать радиус действия оповещения.
